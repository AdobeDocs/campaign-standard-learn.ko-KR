# 데이터 통합 작업의 상태 확인

>[!NOTE]
>
>이 기능은 현재 베타 버전으로, 사전 통보 없이 자주 업데이트하거나 수정할 수 있습니다.
>
>이 기능을 구현하려면 [!UICONTROL Adobe Customer Support]에 연락하십시오.
