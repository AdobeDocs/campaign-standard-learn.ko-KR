# 데이터 암호 해독

Campaign 컨트롤 패널을 사용하면 공개/개인 키 쌍을 사용하여 Adobe Campaign 인스턴스로 들어오는 외부 데이터의 암호를 해독할 수 있습니다.
공개 키를 만들고 인바운드 데이터의 암호 해독을 위해 Campaign 인스턴스로 가져와서 설치하는 방법을 배웁니다.
